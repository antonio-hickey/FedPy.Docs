
# [FedPy.Docs](/)

Web documentation for [FedPy](https://github.com/antonio-hickey/FedPy)

## Run this site locally

- Clone the repo:

```
git clone https://github.com/antonio-hickey/FedPy.Docs.git
```

- Install:

```
cd FedPy.Docs && npm install
```

- Start the server:

```
npm run docs:dev
```

- Run the site in your browser:
 
```
http://localhost:8080/
```
