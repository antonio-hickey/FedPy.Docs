# total()

## About

``` python
FedPy.SOMA().total()
```

This command takes no parameters, and is for aggregating the value of all types of securities held the SOMA portfolio, returning a `float`.

## Example use
``` python
import FedPy

soma_total = FedPy.SOMA().total()
print(soma_total)
```

#### Output:
```
7996833535330.9
```
