# Installation

## Install using pip

Installing FedPy is very simple with just run the following command:

`pip install FedPy`

or using pip3

`pip3 install FedPy`

## Uninstall using pip

You can remove FedPy entirely by running the following commands:

`pip unintsall FedPy`

or using pip3:

`pip3 unintsall FedPy`
